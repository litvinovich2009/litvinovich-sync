# Check Up

>[!quote] *Talking with your doctor about the treatment process can help you relieve the stress.*

### Leukocytes (explanation)
- **lymphocytes - лимфоциты** (их повышение может говорит о вирусной нагрузке)
- **monocytes - моноциты** (может быть повышение при аутоиммунных заболеваниях)  
- **eosinophils - эозинофилы** (показатели аллергических реакций; могут определять наличие паразитов)
- **basophils - базофилы** (показатели аллергических реакций; for example: food, plants, etc.)
- **neutrophils - нейтрофилы**

Первый и базовый этап чекапа будет [[Анализ крови (blood test)#Общий клинический развернутый анализ крови (ОАК) с лейкоцитарной формулой|ОАК - общий анализ крови]].
К нему желательно добавить [[Анализ крови (blood test)#**Blood biochemistry**|биохимию крови]].

Для более подробной информации можно сдать [[Анализ мочи#Total urine analysis (ОАМ - общий анализ мочи)|ОАМ - общий анализ мочи]], а также ее [[Анализ мочи#Biochemistry urine analysis - urinalysis (биохимия мочи)|биохимический анализ]].
 
После этого можно сдать [[Копрограмма|копрограмму]], в нее также входят [[Копрограмма#Stool chemistry|биохимия]] и [[Копрограмма#Stool microscopy|микроскопия]].

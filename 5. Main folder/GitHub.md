# GitHub

## Commands
- `git init` — создание репозитория в текущей директории;
- `git add .` — будут добавлены все файлы в текущей директории и в поддиректориях;
- `git log` — проверка выполненного коммита;
- `git commit` — коммит, где в файловом редакторе нужно написать комментарий (просто комментарий, без ""/#);
- `git commit -m ""` — коммит, где в "" можно указать комментарий (без файлового редактора, как в пункте выше);
- `git status` — проверка изменений;
- `git pull origin main` — "подтягивает" изменения из удаленного репозитория (`origin` — имя моего удаленного репозитория по умолчанию)

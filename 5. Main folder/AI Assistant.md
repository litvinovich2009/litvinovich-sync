## "Мозг" ИИ-ассистента
### Payment
При подключении ChatGPT нужно **пополнить баланс** для использования API ключа.

Большинство компаний предоставляют возможность попробовать AI-ассистентов **бесплатно за свой счет** — они дают $5-10 **на первые 3 месяца**. 
- Для пробных попыток этого может хватить на первое время;

Далее нужно пополнять свой баланс на то кол-во, которые необходимо исходя из ресурсов AI, которые я использую.
- У каждой модели свои тарифы;
- Оплата за 1 запрос к ИИ-ассистенту **достаточно дешевая**;

**Токен** = **4 символа** (+-)

**Оплата** = **сумма** токенов за **вход** (input, входные данные) и **выход** (output, выходные данные)

#### Example
Допустим, модель стоит:
- $0.001 за 1000 входных токенов;
- $0.002 за 1000 выходных токенов;

Input = 500 токенов (2000 символов +-)
Output = 1000 токенов (4000 символов +-)

**Оплата за такой запрос** = (500/1000 * 0.001) + (1000/1000 * 0.002) = $0.0025

# Adjectives' suffixes

## -ing / -ed 
Кто-то дразнит кого-то — **-ing** :
- annoy — annoying person (дразнящий человек);
- interest — interesting film (интересный фильм, он меня заинтересовал);

Тот, кого раздражали — **-ed** (прилагательные с этим окончанием чаще всего описывают эмоции, и из используют касательно людей) :
- annoy — annoyed person (раздраженный человек);
- interest — I'm interested in this film. (Я заинтересованный этим фильмом.)

## -ful / -less
Эти суффиксы — антагонисты друг-другу.

Суффикс **-ful** означает **наполненный чем-то** (быть наделенным чем-то) :
- beauty (красиво) — beautiful (красивый);
- harm (вредно) — harmful (вредный);
- wonder (чудо) — wonderful (удивительный / замечательный);

 Для прилагательного суффикс **-ful** с одной буквой **l**. Суффикс **-fully** используем для образования наречий :
- wonder — wonderfully (замечательно)
- hope — hopefully (надеюсь)

Суффикс **-less** означает **без чего-то** (отсутствие какого-то признака) :
- brain — brainless (без мозгов);
- harm — harmless (безвредный);
- power — powerless (бессильный);

## -able / -ible
Суффикс -**able**, -**ible** указывают на возможность и значение **"выполнимый, способен"**. Прибавляется к глаголам.
- incredible — невероятный;
- impossible — невозможный;

-**Able** используется, когда основа слова может использоваться самостоятельно (notice, read, forget). Если слово оканчивается на согласный, то он удваивается перед -**able**.
- notice — noticeable (заметный);
- passable — проходимый, проезжий;
- advise — advisable (целесообразный);
- read — readable (читабельный);
- forget — forgettable (незапоминающийся);

-**Ible** присоединяется к основам, которые не используются самостоятельно (**possible**, but not just **poss**), или в случае исключения из правил использования -**able**.
- access — accessible (доступный);
- flex — flexible (гибкий);
- audible — слышный;
- permissible — допустимый;
### Советы, как запомнить:
1. **Если не знаешь — пробуй -able.** Оно встречается **в 4–5 раз чаще**, чем -ible.
2. **-ible** часто не имеет "основы" в виде отдельного слова, а **-able** — почти всегда:
	- **enjoy → enjoyable** (глагол есть)    
    - **poss → possible** (такого глагола poss нет!)
## -ous
Суффикс **-ous** выражает характеристику, имеет значение "**быть наполненным каким-то качеством**". Часто прибавляется к существительным.
- danger (опасно) — dangerous (опасный);
- poison (яд) — poisonous (ядовитый);
- mystery (мистика) — mysterious (мистический);
- nerve (нерв) — nervous (нервный);

# Markdown Viewer

Markdown Viewer — это программное обеспечение, предназначенное для **отображения файлов**, написанных на языке разметки **Markdown**. Он позволяет просматривать форматированный текст, не требуя дополнительных инструментов.

Я использую его как **расширение** в браузере **Chrome**.

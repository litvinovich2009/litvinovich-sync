# Adjectives. Number + noun

- **Definitely пишем через дефис** :
	- two-thousand-year period (двухтысячелетний период);

- **Пишем в единственном числе** (не в множественном числе) :
	- three-kilometer road (трехкилометровая дорога);

- **Цифры до 10 пишем словами** (one, two, three ...), **длинные — цифрами** (20, 35, 77 ...) :
	- I'm 15-year-old boy. (Я пятнадцатилетний пацан.)

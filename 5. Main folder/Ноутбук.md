# Ноутбук

Скрытый Агрессивный режим работы для процессора (нужно отключить)

Проблемы с перезапуском
Попробуйте выключить быстрый запуск в настройках электропитания.
- перейдите в панель управления -> электропитание -> выбрать действия при закрытии крышки -> снимите галочку с быстрого запуска

## Как выбрать?
### Экран
OLED - матрица, которая дает глубокий черный цвет
100% sRGB - хорошо (но это база, ниже 80% - х8йня)

#### Яркость дисплея
Средняя норма - 250 кд\м2
Хорошая яркость - 300 кд\м2

# Ускоряем работу в Obsidian

## Складывание вкладок в вертикальную стопку
Чтобы сложить вкладки:
- `Alt + S`
- нажать на стрелочку и выбрать Stack tabs:
	- ![[Pasted image 20250603144605.png|200]]
	- Горизонтальный скролл вкладок — Shift + Scroll
## Передвижение строк вверх / вниз
Строки можно передвигать с помощью хоткеев:
- `Alt + Shift + ↑`
- `Alt + Shift + ↓`
## Закладки
Bookmarks — закладки. Можно добавлять туда заметки, создавать там папки и т.д.

# Анализ мочи

>[!info] Перед забором подмыться, брать порцию с утра, первые 3 - 4 секунды мимо банки, не хранить мочу больше 1,5 часов

## Total urine analysis (ОАМ - общий анализ мочи)
- [k] **Key indicators:**
	- **Color** (light yellow), **very bright** - maybe diabetes, **dark** - dehydration, liver diseases
	- **Clarity** **/ transparency** (**прозрачность**, прозрачная)
	- **Specific gravity** (плотность, 1,003 - 1,005)
	- **pH** (4,5 - 7), **ideally** - 5 - 6, **more alkaline (более щелочная среда, > 5-6)** - many vegetables or fruits in the diet, **more acidic (< 5-6)** - maybe vitamin C

## Biochemistry urine analysis - urinalysis (биохимия мочи)
- [k] **Key indicators:**
	- **Protein** (none), **if it is there** - high-protein diet (many fish, meat, eggs)
	- **Glucose** (none),
	- **Ketones** (**кетоны**, none), **if it’s there** - fasting (голодание), maybe diabetes, carbohydrate deficiency
	- **Bilirubin** (none), **if it’s there** - problems with liver, 
	- **Urobilinogen** (уробилиноген, none)
	- **Erythrocytes** (эритроциты, none - 0 - 3), **ideally** - none, **if they’re there** - kidney stones
	- **Leukocytes** (лейкоциты - WBC, none - 0 - 5), **ideally** - none
	- **Epithelial cells** (**эпителий**, плоский - < 5, переходной - < 1, почечный - none),
	- **Цилиндры** (none)
	- **Bacteria** (none), **if they’re there** - inflammation or incorrect preparation before analysis
	- **Salts / crystals** (none), they can appear because of specific diet (a lot of meat or fruits)

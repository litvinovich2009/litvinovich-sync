# Юридические профессии

## Основные юридические профессии

| Название юридической профессии | Каким законом регламентируется деятельность                                   | Основные задания                                                                                                                                                                                                                                                                 |
| ------------------------------ | ----------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Прокурор**                   | Конституция Украины; "О Прокуратуре"                                          | Поддержание публичного обвинения в суде; руководство досудебным расследованием (розыск, надзор за следователями, дает им поручение, сообщает лицо о подозрении на него)                                                                                                          |
| **Адвокат**                    | Конституция Украины; Закон Украины "Об адвокатуре и адвокатской деятельности" | Представительство лица в суде, защита его прав; предоставление других услуг правовой помощи клиентам                                                                                                                                                                             |
| **Нотариус**                   | Конституция Украины; Закон Украины "О нотариате"                              | Удостоверение прав и фактов, имеющих юридическое значение; предоставление этим документам юридической вероятности                                                                                                                                                                |
| **Следователь**                | Конституция Украины; Криминальный кодекс Украины                              | Государственный служащий, который проводит **предварительное расследование** по уголовному делу. Его задача — **собрать доказательства**, **установить, что произошло**, и подготовить дело для суда (вызывать людей на допросы, проводить обыски, назначать экспертизы и т.д.). |

# Электропитание пк

Электропитание настраивается в **Панели управления** в пункте **Электропитание**.

Сторонние утилиты (*for example: **MSI Center** - моя утилита*) могут сами менять режимы питания (с **Высокая производительность** самостоятельно переключать на  **Сбалансированная**).

>[!important] Лучше снять галочки в *User Scenario / Smart Priority / Performance Optimization* с CPU возле раздела **Режим программного управления**

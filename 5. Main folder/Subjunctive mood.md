# Subjunctive mood

**Условное предложение 2-го и 3-го типов.**

Это воображаемые или нереальные ситуации в настоящем или будущем.

Если в части "If..." используется глагол **to be**, то для всех лиц (I\he\she\it\they\we...) ставится **were**:
- If I were...
- If we were...
- If it were...
- etc.

**Examples**:
- If I were the boss, I would fire him (Если бы я был боссов, я бы уволил его)

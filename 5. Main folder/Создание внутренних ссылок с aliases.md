# Создание внутренних ссылок с aliases

Чтобы преобразовать выделенный мною текст в **alias** к заметке, на которую я хочу сослаться, используется хоткей  `Ctrl + Shift + [`.
- Шаблон для этого хоткея: [[Template - (Link name)]]
- Код: `[[<% await tp.file.cursor(1)%>|<% tp.file.selection() %>]]`
**Результат:** `[[ |some text]]` 

Чтобы преобразовать выделенный мною текст на ссылку **самой заметки**, которую я хочу создать или уже создал, а дальше написать к ней **alias**,
используется хоткей `Ctrl + Shift + ]`.
- Шаблон для этого хоткея: [[Template - (Alias name)]]
- Код: `[[<% tp.file.selection() %>|<% await tp.file.cursor(1)%>]]`
**Результат:** `[[some text| ]]`


>[!info] Эти хоткеи можно настроить с помощью плагина Templater в разделе **Template Hotkeys**

# API

**API** (Application Programming Interface — Программный Интерфейс Приложения) — набор правил и протоколов, определяющих, как различные программные компоненты могут взаимодействовать друг с другом.

![[Pasted image 20250711110405.png]]

![[Pasted image 20250711104959.png|400]]

Это как "язык общения" между **разными приложениями** или **частями одного большого приложения**, позволяющий им обмениваться **данными** и **функциями**.

![[Pasted image 20250711105528.png]]



## Endpoints
**Эндпоинт** (Endpoint) — публичный адрес для конкретной функции.

**Например**:
- marketplace.com==/products==
- marketplace.com==/catalog

Создавать их можно на любом языке с помощью специальных библиотек/[[Термины в IT#Framework|фреймворков]].

# Кронштейн

## Тип крепления
### Ножка
Есть просто кронштейн с **ножкой** (на стол поставить просто).
Правда, это не совсем разумно с точки зрение разгрузки стола.

### Струбцина
Есть **струбцина** (крепление, которое "прикручивается" к поверхности)
- Если стол не позволяет зафиксировать струбцину, можно рассмотреть модели со съемным винтовым упором;

### Настенное крепление
Это для тех, кто точно определился с местом установки монитора (перед тобой должна быть стена).

## Совместимость
Нужно измерить расстояние между отверстиями для крепления у монитора и у кронштейна.

**Стандарты**:
1. 75х75 мм;
2. 100х100 мм;

## Максимальная нагрузка
Нагрузка на кронштейн должна быть на 25-30% больше веса самого монитора без учета подставки.

Также, важно учитывать допустимую диагональ монитора.

## Газлифт
Монитор можно крутить во всех плоскостях.

## Регулировка
Изменение угла наклона требует **предварительного ослабления** винта крепления. После регулировки, затянуть назад.

**ВАЖНО:**
- Нельзя регулировать угол наклона монитора **с зажатым винтом**!

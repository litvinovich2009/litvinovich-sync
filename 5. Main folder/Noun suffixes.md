# Noun suffixes

>[!info] General formula
>Noun / verb / adjective / adverb + **suffixes** = noun

## **Verb** + suffix = noun
Суффиксы для образования существительных из **глаголов**.

1. Могут обозначать **лицо**, которое выполняет это действие.
	- For example: **-ant / -ent**):
		- to assist (помогать / ассистировать) — assistant (ассистент);
		- to consult — consultant (консультант);
		- to study — student (студент);
	- Суффиксы **-or / -er / -eer** употребляются в существительных (object or mechanism), которые **исполняют действие**; чаще всего используются для людей:
		- engineer (инженер)
		- to build — builder (строитель);
		- to train — trainer (тренер);
		- to project (проектировать) — projector (проектор);
		- to sense (чувствовать) — sensor (датчик / сенсор);
2. Могут обозначать **действие** или **его результат**.
	- For example: **-tion, -sion, -ance, -ence, -ment, -al** :
		- to demonstrate — demonstration;
		- to develop — development;
		- to propose — proposal (предложение);
		- to punish (наказывать) — punishment (наказание);
		- to enjoy — enjoyment (наслаждение / удовольствие);
	- Суффиксы **-ery, -ry** указывают на место изготовления, продукцию:
		- to bake (выпекать) — bakery (пекарня);
		- to refine (рафинировать) — refinery (нефтеперерабатывающий завод);

Суффикс **-age** указывает на скопление чего-либо, собирательные понятие или действия:
- to break (ломать) — breakage (поломка, авария);
- to bag (раскладывать по сумкам) — baggage (багаж);
## **Adjective / adverb** + suffix = noun
Суффиксы для образования существительных из **прилагательных и наречий**.

1. Обозначают качества, свойства и характеристики.
	- For example: **-ity, -ness, -cy** :
		- able (способный) — ability (способность);
		- happy — happiness (счастье);
		- careful (осторожный) — carefulness (осторожность);
		- dark — darkness (темнота);
		- urgent (срочный) — urgency (срочность);
		- creative — creativity (креативность);
		- stupidity (тупость);
		- responsibility (ответственность);
## **Noun** + suffix = noun
Суффиксы для образования существительных от **существительных** (ну, типа другое значение).

Профессии — **-ist** :
- novel — novelist;
- geology — geologist;

Cостояние или позиция — **-ship** :
- friend — friendship (дружба);
- lord — lordship (власть / поместье);
- leader — leadership (лидерство);
- owner — ownership (владение / имущество);

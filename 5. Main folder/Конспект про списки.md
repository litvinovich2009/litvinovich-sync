# Конспект про списки

## Определение
**Список (list)** - это коллекция элементов разных типов (int, float, str и др.), которые можно изменять, удалять, добавлять. Элементов в списке может быть сколько угодно. 

>[!example] Вот пару примеров:
> ```python
> numbers=[1,2,3,4,5]
> words=['one','two','three']
> mixed=[1,3,'five',7]
> ```

---
## Индексация списка
В Python каждый элемент списка ассоциируется с **индексом**. Мы можем получить доступ к элементам массива, используя номер индекса (**0, 1, 2,…**).

```python
print(numbers[0]) # 1 (первый элемент списка)
print(words[2]) # three (третий элемент списка)
```


>[!important] **Примечание**
> Отсчет элементов в списке всегда начинается с индекса **0**. Отсюда следует, что первый элемент списка находится под индексом 0, а не 1.


Если мы создадим список с такими элементами: 

```python
spisok=['Red', 'Green', 'White']
```

то их индексы будут такими:

| Элементы списка                                     | 'Red' | 'Green' | 'White' |
| --------------------------------------------------- | ----- | ------- | ------- |
| Индекс (если начинать с начала списка)              | 0     | 1       | 2       |
| Отрицательный индекс (если начинать с конца списка) | -3    | -2      | -1      |


Мы можем так же узнать длину списка с помощью `len()`

```python
print(len(spisok)) # 3 (в списке 3 элементов)
```

## Другие операции со списками (добавление, удаление, замена, сортировка элементов)

### Добавление
Чтобы добавить какой то элемент **в конец** списка, воспользуемся `append()`, а что бы вставить элемент в определенную позицию - будем использовать `insert(индекс, позиция)` 

```python
a=[1,8,3,4,10]
a.append(32) # [1, 8, 3, 4, 10, 32]
a.insert(2, 2025) # [1, 8, 2025, 3, 4, 10, 32]
```


Также можно использовать метод `extend()`, что бы добавить элементы **одного** списка в конец **другого:** 

```python
b=[77, 78, 79]
a.extend(b) # [1, 8, 2025, 3, 4, 10, 32, 77, 78, 79]
```

---
### Удаление
Чтобы удалить один или несколько элементов из списка, можно воспользоваться оператором `del`, а если мы знаем конкретный индекс - `remove()` :

```python
languages=['Python', 'C++', 'Java', 'Rust']
del a[1] # ['Python', 'Java', 'Rust']
a.remove(2) # ['Python, 'Java']

```


Метод `pop()` удаляет элемент списка **с конца**:

```python
a.pop() # ['Python']
```

---
### Сортировка, замена, копия и очистка
Для сортировки используют `sort()` :

```python
numbers=[3,8,10,2,4]
a.sort() # [2, 3, 4, 8, 10] (в порядке возрастания)
a.sort(reverse=True) # [10, 8, 4, 3, 2] (в порядке убывания)
```


Чтобы изменить какой то элемент, можно использовать оператор = :

```python
a[3]=12 # [10, 8, 4, 12, 2]
```


Мы можем создать копию нашего списка с помощью `[:]` :

```python
copy=a[:]
print(copy) # [10, 8, 4, 12, 2]
```


Когда список нужно почистить **полностью**, пропишем `clear()` :

```python
a.clear() # [] - список пуст, чистота и порядок!
```

## List comprehensions
**List comprehension** — это способ быстро создать новый список с помощью цикла в одной строке. 
Синтаксис выглядит так: `new_list=[выражение for элемент in итерируемый_объект]` 

```python
progression=[x**2 for x in range(1,10)]
print(progression) # [1, 4, 9, 16, 25, 36, 49, 64, 81]
```

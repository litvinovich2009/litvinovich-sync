# Дешевые продукты - помощники для здоровья

1. **Имбирь** (помогает желудку вырабатывать **соляную кислоту - HCl**, тем самым улучшает дальнейшее переваривание пищи)
	- несколько слайсов имбиря (можно в каждый прием пищи (for example: lunch and dinner))
2. **Грецкий орех** (качественные жиры: альфа-линоленовая кислота и другие)
3. **Чеснок** (содержит вещество **аллицин**, обладающее противовирусными, противовоспалительными, антибактериальными свойствами; восстанавливает микрофлору кишечника, поддерживает иммунитет, борется с грибковыми образованиями) 
4. **Чернослив** (действует как слабительное - улучшает процесс дефекации) ![[Pasted image 20250424201520.png|200]]
	- лучше брать у бабушек (обычно выглядит как на картинке)
	- блестящий, натянутый, пухленький - скорее всего не натуральный (возможна Е220 и куча сахара)
5. **Оливковое масло** (улучшает процесс дефекации, содержит много мононенасыщенных кислот - w9, содержит vitamin E)
	- Extra Virginie, in dark bottle, unrefined (нерафинированное)
	- Если хранится уже больше 30 дней - частично уже окислилось (лучше брать в небольшом количестве)
6. **Горчица** (стимулирует желчный)
7. **Гранат** (содержит донаторы оксида азота, активизирующие кровообращение ([[Вазодилатация |вазодилатационный эффект]]), улучшают потенцию у мужчин)
8. **Темные ягоды** *(голубика, смородина, черника, ежевика, клюква)* - содержат **биофлавоноиды**, много клетчатки, vitamin C

# UI and UX

**UI/UX дизайн** — это комплексное направление в области проектирования пользовательских интерфейсов, включающее в себя как внешний вид (UI), так и взаимодействия пользователя с продуктом (UX).

**UI** (User Interface — пользовательский интерфейс) — это то, как продукт выглядит (графический дизайн, шрифты, цвета, элементы навигации и т.д.).

**UX** (User Experience — пользовательский опыт) — общее впечатление пользователя при взаимодействии с продуктом (удобство, логика, эффективность и удовлетворенность).

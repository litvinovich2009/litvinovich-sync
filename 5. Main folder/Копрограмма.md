# Копрограмма

>[!info] Не есть за 2 - 3 дня продукты типа: красное мясо (говядина, почки, печень, сердце, язык), красная рыба, молоко)

## Stool test (копрограмма - анализ кала)
- [k] **Key indicators:**
	- **Консистенция** - мягкая
	- **Consistency** - formed / оформленный
	- **Smell** - обычный каловый (зловонный запах - microflora disorder (нарушение микрофлоры), избыточный бактериальный рост (СИБР))
	- **Color** - brown (grey - stomach issue, yellow - gallbladder)
	- **pH** - 6.0 - 8.0, ideally - 7.0 ( < 6.5 - issue with the digestive tract (проблемы с кишечником), > 7 - issue with stomach)
	- **Slime (слизь)** - none (if it’s there in small quantities - work with nutrition, if in big quantities - гастроэнтеролог)
	- **Blood** - none
	- **Остатки не перевариваемой пищи** - none (if it is there - low HCl (низкая кислотность, плохая перевариваемость)

## Stool chemistry
- [k] **Key indicators:**
	- **reaction to protein / hidden blood** (none), **if it’s there** - к гастроэнтерологу
	- **реакция на стеркобилин** (positive)
	- **реакция на билирубин** (negative)

## Stool microscopy
- [k] **Key indicators:**
	- **Мыш. волокна с исчерченностью** (none), **if it’s there** - плохо, непереваренный белок
	- **Мыш. волокна без исч.** (none), **if it’s there** - the same as with point above
	- **Соединительная ткань** (none)
	- **Neutral fat** (none)
	- **Fatty acids** (none)
	- **Salts of fatty acids** - none
	- **Non-digestible fiber** **(не перевариваемая клетчатка)** may be in small quantities
	- **Digestible fiber** (none)
	- **Йодофильная флора** (none), **if it’s there** - СИБР / избыточный грибковый рост в кишечнике
	- **Дрожжевые грибы** (none)
	- **Epithelium** (цилиндрический и плотный, none), **if it’s there in small quantities** - work with nutrition, if in big quantities - гастроэнтеролог
	- **Leukocytes** (none), same as in the point above
	- **Простейшие** (none)
	- **Яйца гельминтов** (none), **if it’s there** - работа с паразитологом**

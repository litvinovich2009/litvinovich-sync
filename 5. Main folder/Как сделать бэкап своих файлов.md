# Бэкап моих файлов на пк

Образ моей системы Windows на данный момент хранится на диске **/D** в папке **/WindowsImageBackup**.

Бекап моих файлов (их резервная копия) хранится в этом же диске **/D** в папке **/FileHistory**
(**/Admin** - старые файлы, **/VladisLove** - новые)

1. Чтобы сделать копию образа системы Windows, в [[00 Панель управления|Панели управления]] выбрать `Резервное копирование и восстановление`.
2. Чтобы сделать резервную копию файлов (через **/FileHistory** в системе), в [[00 Панель управления|Панели управления]] выбрать `История файлов`.

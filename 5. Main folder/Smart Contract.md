# Smart Contract

**Смарт-контракт** — это **технология или программный код**, который может быть написан для выполнения бесчисленного множества задач. 

Kюбой разработчик может написать смарт-контракт и развернуть на совместимом блокчейне.

Каждый раз, когда разработчик создает новое децентрализованное приложение ([[dApps|dApp]]) или новый финансовый протокол ([[DeFi]]), он пишет и развертывает один или несколько **уникальных смарт-контрактов**.
